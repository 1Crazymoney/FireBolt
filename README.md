## Feature

- Lightning Network (LND/LDK/CLN/Greenlight)

- Nostr

- Bitcoin

- HyperCore 

- TBD

- LSP

- CoinJoin 

- Multisig 

- PayJoin

- Bitcoin-only

- Backup using Nostr keys and BIP-39

- Management Nostr Keys

- Contact list via Nostr (NPub) and DID (Web 5)
